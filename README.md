# 題名
生まれてから何日経ったか

# 使い方
$./kadai.py

と入力すると
誕生日を「年-月-日」で入力してください (ex: 2005-12-16)
という表示が出るので入力する
年と月、日の間のハイフンが必要で7月生まれなどの場合は07と入力する必要があります。
未来の日付を指定することはできません

例
誕生日を「年-月-日」で入力してください (ex: 2005-12-16): 2000-01-01
